### PDF 解锁酶 - 解除 PDF 编辑限制的工具

根据规范，PDF 的编辑限制是由软件自行决定是否遵循的。也就是说，技术上可以忽略这个限制。

本软件通过 pikepdf 来批量处理输入的文件，删除所有的编辑限制。

可以直接阅读的 PDF，可以直接删除限制。

阅读需要密码的 PDF，输入阅读密码即可删除限制。
